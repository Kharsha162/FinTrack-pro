# Accessibility

- WCAG 2.1 AA aligned layout and styles.
- Semantic structure: landmarks, headings, lists, tables, charts labelled.
- Keyboard support: focus styles, skip links, interactive elements reachable.
- Screen reader support: ARIA labels, roles, descriptions for dynamic content.
- Forms: labels, hints, validation messages; color-safe error states.
- Tables: headers, roles, consistent focus and readable typography.
- Charts: labelled axes and tooltips; color choices with sufficient contrast.
- Dark/Light modes with contrast-preserving palettes.
